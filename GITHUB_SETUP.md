# GitHub Repository Setup

This document contains manual setup steps you need to complete on GitHub.

## Repository Topics/Tags

Add these topics to improve discoverability:

**To add topics:**
1. Go to https://github.com/quietloudlab/ai-interaction-atlas
2. Click the gear icon ⚙️ next to "About" (top right of repo page)
3. Add these topics:

**Recommended topics:**
- `ai-design`
- `ai-ux`
- `interaction-design`
- `design-systems`
- `taxonomy`
- `ai-patterns`
- `user-experience`
- `product-design`
- `open-source`
- `typescript`
- `react`

**Also set:**
- **Description:** "A shared language for designing AI experiences"
- **Website:** https://ai-interaction.com
- **Check:** ✅ Releases
- **Check:** ✅ Packages

## Enable GitHub Discussions (Optional)

If you want community discussions:

1. Go to Settings → Features
2. Enable "Discussions"
3. Create categories:
   - General
   - Pattern Suggestions
   - Q&A
   - Show and Tell

## Enable GitHub Projects (Optional)

For tracking roadmap publicly:

1. Go to Projects tab
2. Create new project: "Atlas Roadmap"
3. Add columns: Backlog, In Progress, Done
